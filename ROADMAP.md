# ROADMAP

## Current Direction

CRUXpider is evolving from an AI-paper lookup tool into an AI4Science research asset discovery platform.

## Near Term

- Replace binary `ai_related` framing with structured `research_profile`
- Add topic-driven and area-driven research asset discovery
- Improve dataset link quality with stronger public metadata fusion
- Surface reproducibility signals more clearly in the UI
- Harden CI and branch protection for external contributors

## Mid Term

- Add community verification for paper-dataset-code links
- Improve asset deduplication across OpenAIRE, DataCite, Crossref, and OpenAlex
- Add domain-specific taxonomies for materials, chemistry, biology, medicine, climate, and robotics
- Introduce persistent caching and better request observability

## Later

- Build a contributor-backed research asset graph
- Add moderation workflows for community-submitted links
- Expand topic exploration into saved collections and reading lists
